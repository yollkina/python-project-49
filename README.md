Пораскинь мозгами - проверь, насколько хороши твои навыки работы с числами в этих 5 играх, которые можешь запустить прямо в терминале


## Requirements
+ git
+ Python (>=3.10)
+ uv
+ make


## Install
```
make install
make build
make package-install
```
